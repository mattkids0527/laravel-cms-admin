## 風險與注意事項

- Alpine.js 使用 CDN（`3.x.x`），正式環境建議鎖定具體版本號（如 `3.14.8`）
- Tailwind dark mode 為 CDN 即時編譯，`dark:` variant 需確保 CDN 版本支援（目前最新版本均支援）
- 若未來改用 Vite + Tailwind build，需在 `tailwind.config.js` 加入 `darkMode: 'class'` 並移除 CDN 的 inline config
