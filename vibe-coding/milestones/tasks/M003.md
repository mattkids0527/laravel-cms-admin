# M003 — 後台選單權限系統

## 目標

建立資料驅動的後台選單系統，讓管理員可依角色勾選可見的選單項目，側邊欄動態顯示。

## 設計確認

- 選單結構：兩層（群組 + 項目），群組不可點擊，項目對應路由
- admin_menus 內容：由 Seeder 固定，不提供 UI 管理
- 權限設定介面：以角色為主，進入角色頁面後勾選可見選單
- 超級管理員（is_protected=true）自動擁有所有選單權限，不需設定
- 表名：`admin_menus`、`role_admin_menu_permissions`
