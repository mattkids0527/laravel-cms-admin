## 目標

實作後台深色/淺色主題切換、側邊選單收合（icon-only + tooltip），及獨立的外觀偏好設定頁（含一鍵重設）。所有偏好儲存在 localStorage，不需要 DB 欄位異動。

## Tasks

### T1：修改 admin layout（核心前端結構）

- **狀態**：done
- **檔案**：`resources/views/components/layouts/admin.blade.php`
- **內容**：
  1. 加入 Tailwind dark mode 設定（`tailwind.config = { darkMode: 'class' }`，在 CDN 前）
  2. 加入 `[x-cloak]` style 防閃爍
  3. 加入 Alpine.js CDN
  4. 加入 `Alpine.store('appearance', {...})` 腳本（含 init、applyTheme、toggleTheme、toggleSidebar、setTheme、setSidebar、reset）
  5. `<html>` 加 `x-data x-init="$store.appearance.init()"`
  6. `<body>` 加 `dark:bg-gray-950`
  7. `<aside>` 改為動態寬度（`:class` w-64/w-16，transition-all duration-300）
  8. Logo 區加入 sidebar toggle 按鈕（左右箭頭 SVG）
  9. 導覽選單每個項目加入：首字 icon 圓圈 + 文字（展開顯示）+ tooltip（收合 hover 顯示）
  10. 底部使用者資訊收合時隱藏 name/email；登出按鈕加 tooltip
  11. Header 加入 dark mode toggle 按鈕（月亮/太陽 SVG）；加入 dark mode classes
