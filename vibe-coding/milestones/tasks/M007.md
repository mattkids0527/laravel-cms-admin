## 元件使用速查

### 基礎 UI

```blade
{{-- Flash 訊息 --}}
<x-ui.alert type="success" :message="session('success')" />
<x-ui.alert type="error" :message="session('error')" />

{{-- 徽章（color: green/gray/yellow/red/indigo/blue；shape: pill/square） --}}
<x-ui.badge color="green">啟用</x-ui.badge>
<x-ui.badge color="gray" shape="square">角色名</x-ui.badge>

{{-- 按鈕（variant: primary/secondary/danger/link/link-gray/link-danger；size: sm/md/lg） --}}
<x-ui.button variant="primary" href="{{ route('...') }}">+ 新增</x-ui.button>
<x-ui.button variant="secondary" href="{{ route('...') }}">取消</x-ui.button>
<x-ui.button variant="primary" size="lg" type="submit">儲存</x-ui.button>
<x-ui.button variant="link" wire:click="edit">編輯</x-ui.button>
<x-ui.button variant="link-danger" wire:click="delete">刪除</x-ui.button>

{{-- 卡片（padding=false 時改為 overflow-hidden，適合表格外框） --}}
<x-ui.card>內容</x-ui.card>
<x-ui.card :padding="false">無內距卡片</x-ui.card>
```

### 表單

```blade
{{-- 文字輸入（name 用於錯誤顯示；wire:model 等屬性透傳） --}}
<x-ui.form-input label="姓名" name="name" wire:model="name" required />
<x-ui.form-input label="Email" name="email" type="email" wire:model="email" required />
<x-ui.form-input label="密碼" name="password" type="password" wire:model="password" required />

{{-- Select --}}
<x-ui.form-select label="狀態" name="status" wire:model="status">
    <option value="active">啟用</option>
</x-ui.form-select>

{{-- Checkbox（$slot 可放描述說明） --}}
<x-ui.form-checkbox label="記住我" name="remember" wire:model="remember" />
```

### 版面

```blade
{{-- 頁面區塊標題 + 右側按鈕 --}}
<x-admin.page-header title="帳號管理">
    <x-ui.button variant="primary" href="{{ route('admin.users.create') }}">+ 新增帳號</x-ui.button>
</x-admin.page-header>

{{-- 工具列（搜尋 + 篩選 + 操作；wire:model 等屬性透傳至 input） --}}
<x-admin.toolbar placeholder="搜尋帳號..." wire:model.live.debounce.300ms="search">
    <x-slot:filters>
        <select wire:model.live="statusFilter" class="...">...</select>
    </x-slot:filters>
    <x-slot:actions>
        <x-ui.button variant="primary" href="{{ route('admin.users.create') }}">+ 新增</x-ui.button>
    </x-slot:actions>
</x-admin.toolbar>

{{-- 資料表（head/body 必填；footer 選填） --}}
<x-admin.data-table>
    <x-slot:head>
        <th class="px-6 py-3 ...">欄位名</th>
    </x-slot:head>
    <x-slot:body>
        @forelse($items as $item)
            <tr>...</tr>
        @empty
            <tr><td colspan="N" class="px-6 py-12 text-center text-gray-400 dark:text-gray-500">查無資料</td></tr>
        @endforelse
    </x-slot:body>
    <x-slot:footer>
        {{ $items->links() }}
    </x-slot:footer>
</x-admin.data-table>

{{-- 手機底部 sheet（需要父層 <tr x-data="{ open: false }">） --}}
<x-admin.mobile-row-action :title="$user->name" :subtitle="$user->email">
    <x-ui.button variant="sheet-primary" href="{{ route('admin.users.edit', $user) }}">編輯</x-ui.button>
    <x-ui.button variant="sheet-default" wire:click="toggleStatus({{ $user->id }})" @click="open = false">停用帳號</x-ui.button>
    <x-ui.button variant="sheet-danger" wire:click="confirmDelete({{ $user->id }})">刪除帳號</x-ui.button>
</x-admin.mobile-row-action>
```
